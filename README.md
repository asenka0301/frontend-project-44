## Description
Brain-games is a set of console mini games.

## Requirements
* Node.js version 18 or higher.

## Setup
```sh
$ git clone https://github.com/asenka0301/frontend-project-44.git
$ cd frontend-project-44
$ npm ci
$ npm link
```
